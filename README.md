# Haartraum Friseur – Online Terminbuchung

Moderne Friseur-Webseite mit Online-Terminbuchung, Admin-Login, Blog, responsivem Design und sauberer Komponentenstruktur.

## Features

- Moderne Startseite im dunklen/goldenen Salon-Design
- Responsive Header mit Burger-Menü für Handy
- Online-Terminbuchung als Grundstruktur
- Admin-Login
- Admin-Dashboard für Termine
- Blog-Bereich
- FAQ-Bereich
- Scroll-to-Top Button
- CSS Modules
- TypeScript
- Next.js App Router
- Vorbereitung für MySQL, Prisma und E-Mail-Versand

## Tech Stack

- Next.js
- TypeScript
- CSS Modules
- Prisma
- MySQL
- React
- Node.js
